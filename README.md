# jinli
研修として開発するAsanaのクローンアプリ「jinli」

## 名前の由来
- 元となる「Asana」は、ヨガの「アーサナ」が由来となっている。
  - 「アーサナ」とは、「心身のコンディションを整える方法」を意味する。

- それを尊重し、私が習得している太極拳の「勁力（jinli）」を由来とした。
  - 「勁力（jinli）」とは、「全身を協調させて発生する力」を意味する。

## 使用技術
- Vue.js
- Vuex
- webpack(Vue CLI) + Babel
- pug
- Element UI
- Sass(SCSS記法)
- Vue I18n
